# Customer_Shopping_Behavior
Analyzes customer shopping behavior using transactional data.The goal is to uncover insights into spending  patterns, customer segments, product preferences, and subscription behavior to guide strategic  business decisions.

## 🛠️ Project Workflow
1. **SQL:** Extracted raw transaction and customer demographic tables.
2. **Python:** Cleaned missing values, handled outliers, and performed exploratory analysis.
3. **Excel:** Built quick pivot tables for cross-referencing and baseline data checks.
4. **Power BI:** Built the data model and designed an interactive dashboard for tracking metrics like AOV and retention.
5. **PowerPoint:** Summarized the key business recommendations for executive stakeholders.
